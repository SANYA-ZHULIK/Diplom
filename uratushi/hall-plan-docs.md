## Система координат

- **Начало координат**: левый верхний угол зала (100, 100) в пикселях SVG
- **Ось X**: направлена вправо (1 см = 40 пикселей)
- **Ось Y**: направлена вниз
- **Масштаб**: 1:50 → 1 пиксель = 0.0125 м = 1.25 см

### Пример расчёта:
```
Стол T-01: координаты (300, 450)
В метрах: X = (300-100) × 0.0125 = 2.5 м
          Y = (450-100) × 0.0125 = 4.375 м
```

## Требования безопасности

### Ширина проходов

| Тип прохода | Минимум | Факт |
|-------------|---------|------|
| Эвакуационный | 1.2 м | 1.2–2.0 м ✓ |
| Между столами | 1.0 м | 0.8–1.2 м ⚠ |
| К рабочим зонам | 0.9 м | 1.0 м ✓ |

### Пути эвакуации

1. **Основной**: Главный вход → проход → выход (25 м)
2. **Запасной**: Аварийный выход → улица (15 м)

### Противопожарное оборудование

| Оборудование | Количество | Размещение |
|--------------|------------|------------|
| Огнетушители | 2 | Кухня, центр зала |
| Извещатели | 4 | По углам зала |
| Аптечка | 1 | У входа |

## Импорт в CAD

### AutoCAD / NanoCAD / Компас

1. Открыть `hall-plan.dxf` в редакторе
2. Проверить единицы измерения (должны быть мм)
3. Применить масштаб 1:50 при печати

### CorelDRAW / Illustrator

1. Открыть `hall-plan.svg`
2. Разгруппировать элементы (Ctrl+U)
3. Редактировать отдельные объекты

### ArchiCAD / REVIT

1. Импортировать как DWG/DXF
2. Привязать к слоям ARCHICAD
3. Создать зоны автоматически
