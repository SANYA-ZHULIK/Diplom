# 🏛️ У Ратуши - Приложение для бронирования столиков

Веб-приложение для кафе/ресторана "У Ратуши" с системой бронирования столиков и бонусной программой.

## 📋 Функционал

### Для клиентов
- 🔐 Регистрация и вход по email/пароли
- 🗺️ Интерактивная карта зала с цветовой индикацией
- 📅 Бронирование столиков с выбором даты и времени
- 👤 Личный кабинет с историей бронирований
- 🎁 Система бонусных баллов (10 баллов за каждый визит)
- ❌ Отмена бронирования (за 2 часа до визита)

### Для администраторов
- 📊 Панель управления бронированиями
- ✅ Подтверждение/отклонение заявок
- 📍 Отметка визитов (автоматическое начисление баллов)
- 🪑 Управление столиками (добавление, редактирование, позиционирование)
- 👥 Управление бонусами клиентов (ручное начисление/списание)

## 🚀 Быстрый старт

### 1. Настройка Supabase

1. Перейдите в [Supabase Dashboard](https://app.supabase.io)
2. Выберите проект или создайте новый
3. Откройте **SQL Editor** (в разделе Database)
4. Скопируйте содержимое файла `supabase-schema.sql`
5. Выполните скрипт (кнопка Run)

### 2. Назначение администратора

```sql
-- После регистрации в приложении, выполните:
UPDATE profiles SET role = 'admin' WHERE id = 'YOUR_USER_ID';
```

### 3. Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта
3. Перейдите в **Settings** → **Pages**
4. В разделе "Source" выберите "Deploy from a branch"
5. Выберите ветку `main` и папку `/ (root)`
6. Нажмите **Save**

Приложение будет доступно по адресу: `https://ваш-username.github.io/uratushi/`

## 🔐 Безопасность

- RLS (Row Level Security) настроен на всех таблицах
- Пользователи видят только свои данные
- Администраторы имеют расширенный доступ
- Авторизация через Supabase Auth

## 📝 API Endpoints

Приложение использует Supabase JavaScript клиент:

```javascript
// Получение списка столиков
const { data } = await supabase.from('tables').select('*');

// Создание бронирования
const { data } = await supabase.from('bookings').insert([bookingData]);

// Получение бронирований пользователя
const { data } = await supabase
  .from('bookings')
  .select('*')
  .eq('user_id', userId);
```
