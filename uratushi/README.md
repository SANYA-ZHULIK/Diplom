## 🗺️ ПЛАНИРОВКА ЗАЛА

### Основные характеристики

| Параметр | Значение |
|----------|----------|
| Масштаб | 1:50 (1 см = 0.5 м) |
| Размеры зала | 28.0 × 20.0 м |
| Общая площадь | 560 м² |
| Посадочных мест | 60+ |
| Зон | 6 (зал, сцена, бар, кухня, танцпол, санузлы) |

### Зоны помещения

- **Сцена/Подиум** — 72 м², DJ-зона, диско-шар
- **Основной зал** — 350 м², 16 столов, диваны
- **Барная зона** — 60 м², L-образная стойка
- **Танцпол** — 70 м², RGB-подсветка
- **Кухня** — 100 м² (зона персонала)
- **Санузлы** — 22 м², в т.ч. для МГН

### Требования безопасности

- Ширина эвакуационных путей: ≥ 1.2 м
- 2 эвакуационных выхода
- Противопожарное оборудование: 2 огнетушителя, 4 извещателя
- Доступность для МГН: пандус, санузел

### CAD-совместимость

Файлы `hall-plan.svg` и `hall-plan.dxf` поддерживают:
- AutoCAD 2013+
- NanoCAD 5+
- Компас-3D
- ArchiCAD / REVIT (через DXF)
- CorelDRAW, Illustrator
- LibreCAD, QCAD

## 🚀 Быстрый старт

### 1. Настройка Supabase

1. Перейдите в [Supabase Dashboard](https://app.supabase.io)
2. Выберите проект
3. Откройте **SQL Editor**
4. Выполните `supabase-schema.sql`

### 2. Деплой

1. Загрузите файлы на GitHub
2. Settings → Pages → Deploy from branch `main`
